## 《静默轨道：零号苏醒》试玩说明

这是一个**近未来科幻互动剧情**命令行游戏。你将扮演在废弃空间站苏醒的前特工“零”，通过选择影响剧情、角色关系与最终结局。

### 启动方式

在项目目录执行：

```bash
python3 space_station_game.py
```

### 操作说明

- 输入 `1/2/3...`：选择当前剧情分支
- 输入 `status`：查看当前状态变量
- 输入 `save`：保存到默认文件 `savegame.json`
- 输入 `save my.json`：保存到指定文件
- 输入 `load`：读取默认存档
- 输入 `load my.json`：读取指定存档
- 输入 `quit`：退出游戏

### 核心机制

- **状态变量**：
  - `trust_ai`（AI 信任）
  - `trust_human`（人类阵营信任）
  - `memory_integrity`（记忆完整度）
  - `station_stability`（空间站稳定度）
  - `ethics`（价值倾向）
  - `hours_left`（剩余倒计时）
- 每次剧情推进会消耗时间并改变状态。
- 不同状态与关键标记会影响可选分支和最终结局。

### 结局概览

游戏内置多个结局，包括：
- 《灰烬守望者》
- 《无名幸存》
- 《算法加冕》
- 《真相烈焰》
- 《空白之人》
- 《双零悖论》

建议多周目尝试不同立场（信任 AI / 信任人类 / 自我求生）以解锁全部结局。
